IDS Connector API

This is the administrative REST API of the IDS Connector API.

The API provides an administrative interface to manage the IDS Connector at runtime and is used by the default administration dashboard ("web console").

More information: https://helloreverb.com
Contact Info: hello@helloreverb.com
Version: 5.0.0
BasePath:/cxf/api/v1
